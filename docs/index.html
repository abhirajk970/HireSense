Core Capabilities
What HireSense Does
Five tightly integrated modules that automate the entire recruitment lifecycle — from the first application to the final offer.
🤖

AI DSA Interview Engine

Local LLaMA 3 (via Ollama) drives structured DSA interviews through a finite state machine — evaluating intuition, generating progressive hints, and asking follow-up questions dynamically.

LLaMA 3 · Ollama · State Machine
🛡️

AI-Proctored Online Assessment

Secure, invitation-based testing with ML-powered proctoring. Computer vision monitors for suspicious activity during assessments. Auto-MCQ generation from job descriptions.

Computer Vision · Auto-MCQ
📄

ML Resume Screening Engine

FastAPI service using pdfplumber, spaCy NER, and TF-IDF cosine similarity to parse, rank, and match resumes against job descriptions. Extracts CGPA, experience, skills, and organizations automatically.

spaCy · TF-IDF · pdfplumber
🏢

Candidate & Company Portals

Dedicated dashboards for candidates to track applications, and for companies to manage job postings, review AI-generated scores, advance pipeline stages, and schedule interviews.

React · Role-Based Auth
📅

Smart Interview Scheduling

Automated cron-based scheduler sends email reminders at 2-hour and 5-minute marks. Supports both Human and AI interview modes with unique video room IDs per session.

node-cron · Nodemailer · UUID
🔐

Secure Code Execution

Candidate code is evaluated deterministically using the Piston API in an isolated sandbox — with time and memory limits — giving objective, hallucination-free pass/fail results per test case.

Piston API · Docker · Sandbox

AI Interview Engine
How the AI Interview Works
A deterministic state machine wraps LLaMA 3, ensuring reproducible, hallucination-resistant evaluations every time.

Interview State Machine

States progress linearly, preventing LLM chaos and making results auditable.

1
INIT → QUESTION_GIVEN

DSA question fetched from DB; LLM optionally rephrases it in natural language.

2
INTUITION_EXPLAINED → INTUITION_APPROVED

Candidate explains approach. LLM evaluates: correct / partially correct / incorrect. Hint triggered if needed.

3
CODE_WRITTEN → CODE_EVALUATED

Candidate codes in Monaco editor. Piston API executes code against test cases — deterministically, no LLM involved.

4
CODE_EXPLAINED → FOLLOW_UP

LLM asks time complexity, space complexity, and edge case questions. Dynamic follow-ups generated from submission.

5
FINAL_FEEDBACK

Structured report with split scoring delivered to both the candidate and the recruiting company.

Scoring Breakdown

Deterministic code scoring carries the highest weight; LLM-assisted soft-skill scoring supplements it.

Code Correctness (Deterministic)70%
Intuition Quality (LLM)15%
Code Explanation (LLM)10%
Communication (LLM)5%
💡 Hint Penalty
0 hints → no penalty  ·  1 hint → −5%  ·  2 hints → −10%  ·  3 hints → −20%

Progressive Hint Levels

Level 1 — Directional

"Think about reducing repeated work."

Level 2 — Structural

"Can a hash map help here?"

Level 3 — Strong

"Store elements and check complements."

Level 4 — Near-solution

"Check if target − nums[i] exists."

Installation

# 1. Clone repository git clone https://github.com/abhirajk970/HireSense.git cd HireSense # 2. Install Node dependencies for each service cd backend && npm install cd ../oa-backend && npm install cd ../ai-interview-backend && npm install # 3. Set up Python AI service cd ../ai-service python -m venv venv venv\Scripts\activate pip install -r requirements.txt # 4. Pull LLaMA 3 model locally via Ollama ollama pull llama3

Environment Variables

# backend/.env MONGO_URI=mongodb://localhost:27017/hiresense JWT_SECRET=your_jwt_secret_key PORT=5000 # oa-backend/.env MONGO_URI=mongodb://localhost:27017/hiresense_oa PORT=4000

Running All Services

# Terminal 1 — Core API cd backend && npm run dev # Terminal 2 — OA Backend cd oa-backend && npm run dev # Terminal 3 — AI Service (Python) cd ai-service && uvicorn main:app --reload --port 8000 # Terminal 4 — Frontend cd frontend && npm run dev # Terminal 5 — LLM Runtime ollama serve
